# DPM [![Build Status](https://travis-ci.org/dpmtools/dpm.svg?branch=unstable)](https://travis-ci.org/dpmtools/dpm)
A new package manager

### Install
* Run->./install [-b] [-i] [-r] [-m]

-b: backend

-i: installer

-r: repository

-m: manager
